# Optimización de Cadena de Suministro (Enfoque Industrial)

Proyecto de optimización de la cadena de suministro para una planta de manufactura, utilizando análisis estadístico, limpieza de datos con Pandas, regresión múltiple y AutoML para predecir retrasos y optimizar inventarios.

## Objetivo

Predecir retrasos en entregas y optimizar niveles de inventario para una planta de manufactura (estilo Schneider Electric/Arcomex).

## Dataset

*   [Logistics and Supply Chain Dataset](https://www.kaggle.com/datasets/datasetengineer/logistics-and-supply-chain-dataset)

## Competencias Cubiertas (Trayectoria Citizen Data Scientist)

*   **Fundamentos de Estadística para Ciencia de Datos**
    *   Análisis de variabilidad en tiempos de entrega utilizando la distribución normal.
*   **Panda y Numpy en Python**
    *   Limpieza y manipulación de más de 85,000 registros utilizando Pandas.
*   **Análisis Causal y Predictivo Utilizando Regresión**
    *   Regresión múltiple para identificar factores causales del Lead Time.
*   **Desarrollo de Machine Learning Utilizando AutoML/Augmented ML**
    *   AutoML con PyCaret para predecir la probabilidad de disrupción en la cadena de suministro.
